공지사항

저희 CoUzleIT는 프로그래밍에서 꼭 필요한 code 작업을 재밌고 즐겁게 만들고, 교육하는 기업입니다.저희 CouzleIT에서 교육하는 과목으로는 C / C++, 웹표준, 모바일웹앱, java/jsp가 있으며, 프로그래밍을 전문으로 하여 초보에서 부터 전문가까지 쉽고 빠르게 현장에서 충분한 능력을 펼칠 수 있도록 최선의 교육을 지향합니다.
